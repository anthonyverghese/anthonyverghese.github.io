Game Predictor Websitegamepredictor.github.io

I've always wanted to blend my interests in software engineering and basketball, and this site did just that. I created a site that gives a personal touch to basketball fans, and gives them a better way to analyze upcoming games, with graphs and detailed explanations of certain statistics. To complete this website, I utilized web scraping with BeautifulSoup/Python, the D3 library, AngularJS, and HTML/CSS/JavaScript.

Submissions Scenarios Website

Over the summer, I interned at Booz Allen Hamilton with a web application team that focuses on online capital planning for eighteen government agencies. I created a website that asks the government agencies using Booz Allen's web application a series of questions and determines what submission scenario they are dealing with. I utilized HTML/CSS/JavaScript to develop an optimal algorithm, and used Materialize for the UI components.

Resource Pathfinder

For my final project in my data structures class at Cornell, I had to create an algorithm that allows an avatar to escape from a 2D maze with as much gold as possible. I used a min heap implementation with distance from tile/amount of gold at tile as the values put into the heap. Djikstra's algorithm and depth first search were also used to get Indiana Jones to escape on time.
